# Do you want to learn React Actual or migrate to working with Hooks?

This course aims to take you from zero knowledge of React to a competitive level in today's work environment. This course is built 100% in Hooks and functional components.

It also has sections on unit and integration testing in each application that we do in the course, this includes testing components, hooks, customHooks, context, Redux, Fetch, mocks, spies, snapshots and much more related to unit testing.

Without forgetting that we all start from scratch at one point, the course has an introduction to the concepts that we will need to get into React in the most comfortable way possible.

This course has more than 400 lessons and more than 40 hours of video doing different applications and tests that will help us achieve the goal of mastering React and other technologies that surround it, there are many tasks and exercises that will help you assimilate the necessary knowledge to be a FullStack Developer using React and Node in the Backend along with Mongo and Firestore as databases.

And much more!

This course is for all people who want to learn React, want to go from Class Base Components to Functional Components, want to specialize in testing, get into Redux, Firebase, Context or simply learn something new completely and it works.
